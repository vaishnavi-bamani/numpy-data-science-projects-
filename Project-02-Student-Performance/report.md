# 📚 Student Performance Analysis using NumPy

## Project Report

**Project Name:** Student Performance Analysis using NumPy

**Technology Used:** Python, NumPy, Jupyter Notebook

**Dataset:** Student Marks Dataset

**Author:** Vaishnavi Bamani

---

# 1. Introduction

Student performance analysis is an important application of data analytics in the education sector. By examining students' marks, educators can identify performance trends, recognize high achievers, detect students who need additional support, and make data-driven academic decisions.

In this project, a dataset containing marks of students in three subjects was analyzed using the NumPy library. Various statistical calculations and array operations were performed to generate meaningful insights about student performance.

The project demonstrates how NumPy simplifies numerical computation and enables efficient analysis without relying on traditional Python loops.

---

# 2. Project Objectives

The primary objectives of this project are:

* Load student marks data from a CSV file.
* Explore the dataset structure.
* Extract individual subject marks.
* Calculate total and average marks.
* Identify highest and lowest performers.
* Classify students based on grades.
* Determine pass and fail counts.
* Analyze subject-wise performance.
* Generate a summary dashboard using NumPy.

---

# 3. Dataset Description

The dataset consists of marks obtained by students in three subjects.

| Column     | Description                       |
