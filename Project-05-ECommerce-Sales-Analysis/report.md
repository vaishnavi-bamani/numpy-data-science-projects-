# 📊 E-Commerce Sales Analysis Report

## Project Overview

This project analyzes e-commerce sales data using NumPy. The analysis focuses on revenue generation, discount calculations, category-wise sales performance, and overall business insights.

---

## Objectives

- Load and analyze an e-commerce sales dataset.
- Calculate gross and net revenue.
- Analyze discounts.
- Perform category-wise sales analysis.
- Identify high-value orders.
- Generate business insights using NumPy.

---

## Dataset Information

- Total Records: 30 Orders
- Features:
  - Order ID
  - Product ID
  - Category
  - Quantity
  - Price
  - Discount

---

## Analysis Performed

- Dataset Exploration
- Revenue Calculation
- Discount Analysis
- Net Revenue Calculation
- Revenue Statistics
- Category-wise Sales
- High Value Orders
- Revenue Classification
- Top Five Revenue Orders

- Sales Dashboard
