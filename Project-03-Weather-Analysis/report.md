# 🌦️ Weather Data Analysis using NumPy

## Project Report

**Project Name:** Weather Data Analysis using NumPy

**Technology Used:** Python, NumPy, Jupyter Notebook

**Dataset:** 30-Day Weather Dataset

**Author:** Vaishnavi Bamani

---

# 1. Introduction

Weather data analysis helps us understand climate conditions by studying various weather parameters such as temperature, humidity, rainfall, and wind speed.

In this project, a weather dataset containing observations for 30 consecutive days was analyzed using the NumPy library. The objective was to perform statistical analysis, classify weather conditions, and generate meaningful insights without using traditional Python loops.

NumPy's vectorized operations make numerical analysis faster, efficient, and easier to understand.

---

# 2. Project Objectives

The primary objectives of this project are:

* Load weather data from a CSV file.
* Explore the dataset structure.
* Extract individual weather parameters.
* Analyze temperature statistics.
* Analyze humidity levels.
* Classify rainfall conditions.
* Analyze wind speed.
* Categorize daily temperatures.
* Perform statistical analysis using NumPy.
* Generate a comprehensive weather dashboard.

---

# 3. Dataset Description

The dataset contains weather observations collected over a period of 30 days.

The dataset consists of five columns:

| Column      | Description            |
| ----------- | ---------------------- |
| Day         | Day Number             |
| Temperature | Daily Temperature (°C) |
| Humidity    | Relative Humidity (%)  |
| Rainfall    | Daily Rainfall (mm)    |
| Wind Speed  | Wind Speed (km/h)      |
