# 🖼 Image Processing using NumPy Report

## Project Overview

This project introduces basic image processing techniques using only NumPy. A grayscale image is represented as a two-dimensional array, and several common image manipulation operations are performed.

---

## Objectives

- Load grayscale image data.
- Analyze image properties.
- Perform image transformations.
- Apply brightness and contrast adjustments.
- Normalize image data.

---

## Dataset Information

- Image Size: 8 × 8 Pixels
- Pixel Range: 0–255
- Image Type: Grayscale

---

## Image Processing Operations

- Image Statistics
- Brightness Enhancement
- Brightness Reduction
- Contrast Enhancement
- Negative Image
- Binary Thresholding
- Image Cropping
- Horizontal Flip
- Vertical Flip
- 90° Rotation
- Image Normalization
- Image Dashboard

---

## NumPy Functions Used

- np.genfromtxt()
- np.max()
- np.min()
- np.mean()
- np.median()
- np.std()
- np.clip()
- np.where()
- np.fliplr()
- np.flipud()
- np.rot90()

---

## Key Learning Outcomes

- Understanding images as arrays.
- Pixel-level operations.
- Image enhancement techniques.
- Computer vision preprocessing.
- Data normalization.

## Conclusion

This project establishes a strong foundation in image processing using NumPy and prepares learners for advanced computer vision libraries such as OpenCV and deep learning frameworks.
